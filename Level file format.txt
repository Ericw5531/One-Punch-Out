OPO Levels File Format

Header
--------------------
<4 bytes>".OPO"
byte 4 - Background Number
byte 5 - Width in Chunks
byte 6 - Height in Chunks
byte 7 - playerSpawnPosition.x tile
byte 8 - playerSpawnPosition.y tiles

layer 1 - Solid layer
--------------------------------
<byte numOfSolids, byte solidID>

Solid Layer Types:
0 - No Collision
1 - Solid
2 - Slow
3 - Slippery
4 - Death
