OPO Levels File Format

Header
--------------------
<4 bytes>".OPO"
byte 4 - Width in Chunks
byte 5 - Height in Chunks
byte 6&7 - {chunk, remainder tile} (x position) playerSpawnPosition.x
byte 8&9 - {chunk, remainder tile} (y position) playerSpawnPosition.y

layer 1 - Solid layer
--------------------------------
<byte numOfSolids, byte solidID>

Solid Layer Types:
0 - No Collision
1 - Solid
2 - Slow
3 - Slippery
4 - Death
